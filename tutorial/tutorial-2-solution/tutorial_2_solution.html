Define a new object class Car which represents a car in terms of the owner’s name (attribute owner), the model of the car (model), the mileage (kilometers) and amount of gasoline (gas) at the recorded fuel stops. Generate two new instances for Mike’s and Jenny’s car, assign attributes and access them using pythons’s dot notation.

  2. Implement a method print_car() that describes the object in a message saying something like: "This is Mike's car. It is a Opel Manta.".

  3. Add a method avg_consumption() that returns the average consumption (in liter per 100 kilometers) between the recorded fuel stops.

  4. Implement a new method add_stop() that allows to add data from a new fuel stop, i.e., enter a new entry for kilometers and gas. Add a new fuel stop and check if it is successfully listed in the output of the avg_consumption() call.
